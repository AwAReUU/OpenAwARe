Detailed Description

A singleton that holds a connection with the server. This class can be used to login and register on the server. It can also send requests to the server. The Post() and Get() methods are the easiest methods to send a Request.

For example:

Client.Init("localhost:8000", new User(..));
string search = "{ 'input': 'orange' }";
Client.Post("/ingredients/search", search)
.Then(res => Debug.Log(res.Get("list")))
.Catch(err => Debug.LogError(err)) // Optional
.Send();
